# 环境支持

yp-ui是以Vue3和Element-plus为基础， 所以可以在支持 ES2018 和 ResizeObserver 的浏览器上运行。

由于 Vue 3 不再支持 IE11，yp-ui 也不再支持 IE 浏览器。

# 使用包管理器

# 选择一个你喜欢的包管理器

# NPM
$ npm install yp-ui --save

# Yarn
$ yarn add yp-ui

# pnpm
$ pnpm install yp-ui

如果网络环境不好，建议使用相关镜像服务 cnpmopen in new window 或 中国 NPM 镜像。
